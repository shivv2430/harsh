Course Curriculum: Web Technologies

This Web Technologies course introduces students to the fundamentals of HTML, CSS, and JavaScript. The course emphasizes building responsive and interactive web pages using modern tools and techniques. It is essential for all Computer Science and Information Technology students.

Course Overview

The course is divided into multiple modules that cover web design and development concepts. Each module includes theory, practical exercises, and projects to enhance learning. Note: HTML uses special characters like <, >, and & for tags and entities.

Course Modules (Ordered List)

  1. Introduction to Web Technologies
  2. HTML: Structure of Web Pages
  3. CSS: Styling the Web
  4. JavaScript: Making Web Pages Interactive
  5. Projects and Case Studies

Technologies Used (Unordered List)

  • HTML5
  • CSS3
  • JavaScript (ES6+)
  • Bootstrap Framework

HTML Tags Description (Description List)

<h1> - <h6>
Used for headings of different sizes.
<p>
Defines a paragraph of text.
<ul> and <ol>
Used for unordered and ordered lists, respectively.
